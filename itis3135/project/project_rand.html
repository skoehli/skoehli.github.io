  • Page Designs:
    • Homepage/Introductory page: This page will welcome users to the site and tell them what the site is about. Will include information on why the pepper garden exists, why homemade sauces are being made, and background on the garden. Users will not be required to enter any information on the page. As such, there will be no forms, fields, dropdown menus, or buttons. There will also be no hyperlinks aside from the navigation menu and footer.
    • Pepper information page: This page will tell users about each pepper plant: How long it takes to grow, what ripe peppers look like, and how hot the peppers are. Each pepper will have its own section with a picture and caption. Users will not be required to enter any information on the page. As such, there will be no forms, fields, dropdown menus, or buttons. There will also be no hyperlinks aside from the navigation menu menu and footer.
    • Sauce page: This page will tell users the ingredients used to make sauces. Does not include specific recipes. Users will not be required to enter any information on the page. As such, there will be no forms, fields, dropdown menus, or buttons. There will also be no hyperlinks aside from the navigation menu menu and footer.
    • Pepper calculation page: This page will inform users how many peppers they will need for a batch of sauce. There will be a field to enter in the number of bottles they wish to make in the batch, as well as a dropdown menu that displays different sizes of bottles. Lastly, the user must choose a sauce they would like to make, which will be chosen from another drop-down menu. After the user has entered the data, they can click a submit button and the data will be processed. Upon clicking the button, a JavaScript function will tell the user how many peppers they need for that batch. The only user input that will require validation is the number of bottles field: only numbers are allowed.
    • Recipe page: This page helps readers find ideas to use their sauces on by providing some recipes to follow. Photos will accompany each recipe, along with lists of ingredients used in those recipes. Users will not be required to enter any information on the page. As such, there will be no forms, fields, dropdown menus, or buttons. There will also be no hyperlinks aside from the navigation menu menu and footer.
  • Dynamic functionality: The Pepper Calculation page will use JavaScript in order to calculate the number of peppers a user needs to make a batch of sauces of a particular type. JavaScript will be used here specifically because a function can be easily made to do such a calculation and display information on the page that show the results.
